# 📚 Online Learning Journal System

A full-stack web application designed to digitize the student learning journal process.This system replaces manual, paper-based tracking with a secure, centralized digital platform.

---

##  Project Overview
In modern education, continuous reflection is vital for academic growth.This application streamlines the process by allowing students to document their weekly progress and enabling faculty to provide efficient feedback.

##  Technology Stack
* **Frontend**: HTML5, CSS3, and JavaScript.
* **Backend**: Node.js and Express.js for robust RESTful API routing.
* **Database**: MongoDB Atlas for secure, cloud-hosted storage.
* **Utilities**: Postman for API testing and VS Code for development.

##  System Architecture
The project follows a three-tier architecture to ensure scalability and data reliability
1. **Presentation Layer**: Responsive frontend interface.
2. **Application Layer**: Backend logic, validation, and processing.
3.**Data Layer**: MongoDB Atlas for persistent storage.
